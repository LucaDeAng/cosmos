## Extraction Patterns

### Price Patterns
- "€XX" / "EUR XX" / "XX euro"
- "XX/mese" / "XX/anno" / "XX mensili"
- "da €XX" / "a partire da"
- "XX/utente" / "XX/postazione"

### Date Patterns
- "dal XX/XX/XXXX" / "entro il XX/XX"
- "durata XX mesi" / "validità XX anni"
- "Q1 2024" / "H2 2025"
- "entro fine anno" / "prossimo trimestre"

### Status Patterns
- "attivo|active|in uso|operativo"
- "in corso|ongoing|in progress"
- "completato|done|finished|concluso"
- "pianificato|planned|previsto"
- "sospeso|paused|on hold"

### Priority Patterns
- "critico|critical|urgente|emergency"
- "alta|high|importante|prioritario"
- "media|medium|normale|standard"
- "bassa|low|opzionale|nice to have"
